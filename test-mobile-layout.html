Test de Layout Mobile

Esta página es para probar que el botón de admin (candado) no se superpone con el botón del menú móvil.

Instrucciones de prueba:

  1. Redimensiona la ventana del navegador para simular diferentes tamaños de pantalla
  2. En móvil (≤768px): El botón de admin debe aparecer debajo del header (top: 90px)
  3. En tablet (≤1024px): El botón de admin debe estar ligeramente desplazado (right: 70px)
  4. En desktop (>1024px): El botón de admin debe estar en su posición normal (top: 20px, right: 20px)

Comportamiento esperado:

  • El botón del menú móvil (hamburguesa) está en la esquina superior derecha en móviles
  • El botón de admin (candado) no debe superponerse con el menú
  • Ambos botones deben ser fácilmente accesibles
  • La transición entre tamaños debe ser suave
