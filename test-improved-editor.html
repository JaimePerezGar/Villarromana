Editor Controls

In Edit Mode:

  • Click any text to edit it directly
  • Select text to see formatting options
  • Use the + buttons between blocks to add new content
  • Drag blocks by their handle (⋮⋮) to reorder
  • Click images to replace them
  • Use toolbar buttons for global actions

Toolbar Buttons:

  • Add Block: Choose from paragraph, headings, lists, quotes
  • Add Image: Upload or drag & drop images
  • Undo: Revert all changes
  • Save: Save changes to the HTML file
  • Cancel: Exit without saving
